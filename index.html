Assignment #2 - Summarized Semantic Elements

I figured it would also be appropriate to incorporate the Summarized Semantic Elements assignmen to this page.

Semantic elements are used by developers to clearly communicate to browsers what the authors are intending to display. These elements include <header>, <nav>, <aside>, <section>, <article>, and <footer>. They are visible on websites unlike non-semantic elements, such as <div> and <span>. Each semantic element provides a specific function according to its defined purpose. This helps structure and format each section so that developers can customize websites to fit their designs.
